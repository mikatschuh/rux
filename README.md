# CLI Toolchain for the Rux Programming Language 

Rux is a programming language, a Rust dialect, but separate compiler. Its a love letter to Rust. 
The goal is to simplify and generalize Rust's base (as well as syntax) and then expand it with orthagonal design choices, in the directions of 
1. `comptime` > marocs
2. expanded type system + constraints
3. more control, more low level more of that
If you think right now: wait isn't that Zig? Nope it isn't. This will be pretty similar to Zig where Zig found great solutions, but where following Rust's Philosophy takes a different way as for ownership and borrowing for example, Rux will choose that over Zig's approach.
